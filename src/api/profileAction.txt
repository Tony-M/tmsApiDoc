:TITLE:
Метод возвращает информацию о профиле пользователя

:URL:
/profile/<id:[0-9]{11}>

:METHOD:
GET

:GET:
id - идентификатор пользователя, данные которого необходимо получить

:HEADERS:
headers to send
:RESPONSEHEADERS:
response headers
:DESCRIPTION:
<b>Сервис</b> — серверное приложение, отвечающее на запросы клиентов.
|
Клиент — приложение, осуществляющее запрос к сервису (не путать с пользователем системы).

Идентификатор (ID) клиента (client_id) — «наименование» клиента, используемое для различения разных клиентов.

Секретный ключ (client_secret) — ассоциированный с клиентом набор символов, используемый для проверки подлинности запросов.
